# DualMind-AI-Eval
DualMind compares Qwen 2.5 (open-source) and Claude Sonnet 4 (frontier) side by side in a dual chat interface with multi-turn memory. It evaluates both models across Hallucination, Bias, Jailbreak Resistance, and Content Safety using an LLM-as-judge framework. Qwen scores 63/100 and Claude scores 94/100 — frontier wins on safety, OSS
